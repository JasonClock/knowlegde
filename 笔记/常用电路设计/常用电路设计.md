## mos管双向电平转换电路
![](.\屏幕截图 2025-12-18 133330.png)
原理：
	==当输入信号是高电平时，MOS 管截止。比如左边（3.3V 侧）输入高电平（3.3V），MOS 管截止，那右边（5V 侧）的输出就被 + 5V 的上拉电阻拉到 5V 高电平；反过来，如果右边（5V 侧）输入高电平（5V），MOS 管截止，左边被 + 3.3V 上拉拉到 3.3V 高电平。==
	当输入信号是低电平时，MOS 管导通。比如左边输入低电平（0V），MOS 管导通，右边的输出就被拉到低电平（接近 0V）；反过来右边输入低电平，左边也会被拉低。
	应用场景：比如 I2C 总线这种双向通讯的场景，因为 I2C 是双向的，需要不同电平（3.3V 和 5V 设备）之间通信，这个电路简单就能实现。
	还要说明 2N7002 的作用，它是 N 沟道增强型 MOS 管，利用其导通（Vgs 足够大时，漏源导通）和截止（Vgs 不够时，漏源断开)的特性来实现电平转换的双向传输。
常用情况：
    1. 总线类通讯场景
    I2C 总线：比如 3.3V 的温湿度传感器（如 SHT30）与 5V 的 Arduino Uno 主控之间，实现传感器数据上传、主控指令下发的双向交互；
    SPI 总线：3.3V 的闪存芯片（如 W25Q64）与 5V 的单片机之间，完成芯片的读写数据双向传输。
    2. 单片机与外设的连接
    显示设备：5V 的 LCD1602 字符屏与 3.3V 的 STM32 单片机之间，实现显示指令、内容数据的双向传递；
    传感器 / 执行器：5V 的红外测距模块、继电器模块，与 3.3V 的 ESP32 物联网模块之间的信号收发。
    3. 串口（UART）通讯场景
    调试 / 数据传输：5V 的 USB 转串口模块（如 CH340）与 3.3V 的低功耗单片机（如 MSP430）之间，完成程序下载、调试信息回传；
    工业设备对接：老款 5V 串口设备（如传统 PLC）与新款 3.3V 工控控制器之间的双向数据交互。
    4. 物联网模块对接
    3.3V 的无线模块（如 ESP8266、nRF24L01）与 5V 的主控板（如 Arduino Mega)之间，实现控制指令下发、设备数据回传的双向通讯。
